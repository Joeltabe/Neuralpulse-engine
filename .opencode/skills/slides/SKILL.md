# Slides

Strategic HTML presentation design with data visualization.

## When to Use

- Marketing presentations and pitch decks
- Data-driven slides with Chart.js
- Strategic slide design with layout patterns
- Copywriting-optimized presentation content

## Knowledge Base

| Topic | Description |
|-------|-------------|
| Layout Patterns | Slide layout templates and variants |
| HTML Template | Complete slide HTML structure |
| Copywriting Formulas | PAS, AIDA, FAB, and more |
| Slide Strategies | Deck structures, emotion arcs, sparklines |

## Requirements

All slides MUST:
1. Use CSS variables from design tokens
2. Use Chart.js for charts (not CSS-only bars)
3. Include navigation (keyboard arrows, click, progress bar)
4. Center align content
5. Focus on persuasion/conversion
